# holodex.enspiral.com

instance of [Holodex](https://github.com/open-app/holodex) for [Enspiral](http://enspiral.com).

## how to

### install

```bash
git clone https://github.com/holodex/holodex.enspiral.com
cd holodex.enspiral.com
npm install
```

### run

```bash
npm start
```

### deploy

```bash
npm run add-remote
npm run deploy
```
